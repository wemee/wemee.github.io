產品功能與特色

1.專為中小企業量身訂做的國際品牌ERP

系統 最適合中小企業所需要，功能模組最完整的ERP,CRM,SCM,BI功能，系統客製修改最容易，導入最快速，整體效益最高的商業應用資訊系統。

2.系統架構採多層式分散架構

區分為前端使用者界面、中間應用系統伺服器與後端資料庫，使用者端程式輕盈，所有系統 交易及運算，以及企業邏輯與系統程式的儲存放置都集中在應用程式伺服器，系統容易維護更新並確保一致性，產品安裝與導入比較簡單。

3.與使用者端Microsoft Office等軟體及各種微軟伺服器系統、視窗作業系統整合度極高。

Navision 操作環境與安裝方式極接近使用者最熟悉的Microsoft Office系統，並與Microsoft Windows XP/Windows 2003最相容。

4.系統整體客製建置最容易

微軟全球技術開發團隊全力支援，系統版本更新升級速度最快。
Navision採物件導向設計開發的物件，內建強大開發環境，所有類型物件統一介面開發。
可輕易讀取修改系統程式原始碼，極易依照。

5.強化企業決策支援

微軟全球技術開發團隊全力支援，系統版本更新升級速度最快。
內建商業智慧(BI)分析工具，極容易建立關鍵績效指標(KPI)，可立即連結所有關聯細部交易紀錄，強化企業整體決策品質與速度。

6.高度整合企業內外部資料庫與應用系統

更強大的通知與警示功能，即時掌握企業內外部、電子商務前後台與供應鏈上下游資訊。

基於開放的 Linux KVM Hypervisor發展與運行。支持Virtual Machine 的生命週期(建立、開機、關機、遷移)管理(VMM,Virtual Machine Management)。以網頁遠距操作，不用進機房就可以維護運算資源。透過 VM Cluster設定與API使用，可讓應用軟體彈性擴充VM資源數量。
