  • 雲端報告管理服務

    雲端客戶拜訪報告管理系統，提供在組織內留存日常跟客戶或夥伴互動的重要資訊標籤設計，提供方便查詢、個人化分類的檢索方式同時方便以人、客戶/夥伴、時間等不同角度查詢拜訪報告以客戶為中心、或以產品為中心的整合角度檢視相關訊息在預設的報告書寫框架下，讓報告者能有所依循，方便撰寫與閱讀。

產品功能與特色

1.交辦工作戰力超靈活

檢視報告後可快速轉寄相關人員，所需欄位可以個別挑選，資安保護貼心週到，轉寄記錄與流向完向完全留存，隨時掌握。

2.聚焦重要客戶不漏失

業務員填寫時可自行定義重要度指標，老闆可藉此聚焦在重要且緊急的報告與客戶，第一時間立即檢視營運成果。

3.訂單搜尋再也不頭大

以產品、客戶為中心的自動標籤設計，更容易追蹤與尋找報告，進階搜尋條件多樣化，連附件內文都可以輕鬆找到。

4.版本清楚控管有依據

所有報告變動都會自動記錄，檢視版本更動狀況一覽無遺，確實掌握報告時間軌跡，讓業務推展過程都有憑有據。

5.行動支援，隨時掌握

Vis-a-Vis®同時支援iOS與Android系統的智慧型手機，不管您用的是iPhone、iPad、HTC等行動裝置，皆可以下載安裝行動版的Vis-a-Vis®

6.隨時不錯過任何商機

不管在辦公室開會或拜訪客戶的路途中，您都可以透過愛報告迅速的閱讀找到您需要的報告，在與客戶拜訪的過程中，更能立即錄音、拍照、錄影，即時的紀錄上傳，同一時間，辦公室的老闆或同仁已經看到您的報告了。

7.行動辦公效率升

不怕客戶拜訪報告片尋不著，在你需要資料時，只要連得上網路，就可以透過智慧行動裝置立即查詢報告。也可以利用行動裝置內建的錄音、錄影、拍照等功能隨時記下資料並上傳，讓共享更及時，互動無距離。

8.協同互動好幫手

協助企業隨時掌握業務資訊、及時回覆進度，使用者可利用重要指標與待辦設定輕鬆瞭解業務歷程並且進行跟催，提升整體工作效率！團隊成員可透過此共享平台，留言提問、共享以及回覆，藉此挖掘組織隱性知識激發創新。

基於開放的 Linux KVM Hypervisor發展與運行。支持Virtual Machine 的生命週期(建立、開機、關機、遷移)管理(VMM,Virtual Machine Management)。以網頁遠距操作，不用進機房就可以維護運算資源。透過 VM Cluster設定與API使用，可讓應用軟體彈性擴充VM資源數量。
