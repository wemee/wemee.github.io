  • 新一代雲端郵件通訊平台

    電子郵件系統是企業對內對外最重要的傳送訊息管道之一，YouCloud Mail不僅提供簡單的操作介面，更能將企業需求的協同作業功能整合到郵件系統內，大幅縮短企業的溝通時效，提高公司整體效益。

產品功能與特色

1.Web化管理介面

提供人性化 Web GUI 管理介面，只需透過瀏覽器就能輕鬆管理 Mail Server，不需專業教育訓練，非專業 IT 人員也能輕鬆管理！

2. 系統備份

系統支援高效能信件儲存技術，並具有郵件備份機制，輕鬆將郵件資料備份到其他主機或備援郵件主機上！

3. 病毒引擎過濾

提供 Clam AV 防毒引擎，可自動偵測到約 128 萬種以上的病毒、蠕蟲、木馬程式...等，且系統會每日自動透過網路更新病毒碼！

4.權限管理機制

提供多人管理及完整的權限分配，管理者視使用者權限開放後台功能給使用者端使用，以控管郵件系統及公司內部機密郵件安全。

5. 專屬網域名稱

不同於一般容易漏信及退信的免費信箱，企業建置私有郵件伺服器，擁有專屬企業的郵件位址，能打造深刻專業形象，也可大幅降低收發信件效率低的難題！

6.垃圾信件過濾

系統提供智慧型廣告垃圾郵件過濾規則，採用分數比例方式判別郵件性質，配合黑白名單設定，就能做出詳盡的郵件過濾報告及多樣性的郵件處理方式！且提供隔離信通知使用者，可各自取回郵件。

7. 郵件閘道器

提供郵件過濾閘道功能，可作為企業內部阻擋垃圾信或病毒信件的先鋒，閘道器過濾對企業有危害信件後，再將乾淨的信件傳送到世界各地的郵件伺服器中！

8. 系統報表日誌

系統提供報表可統計 CPU 主機負載量、網路流量、磁碟使用空間...等，可同時統計信件特定期限範圍內的收發狀況，且以圖表方式顯示系統狀態！

9. DDOS主機防護

支援POP3、IMAP、HTTP...等多種協定，避免主機被暴力猜測導致主機服務停擺 。

10.多網域功能

系統具有多網域功能 ( 共同帳號 )，除本身設定的網域名稱外， 使用者可設定同時接收其他網域名稱信件，讓企業郵件可同時擁有多個網域別名！

11. 支援多種收信軟體

支援標準 POP3、SMTP、IMAP...等多種郵件收信軟體，亦同時支援SSL連線加密技術，提供企業高度靈活應用性的郵件伺服器軟體！

12. 郵件紀錄查詢

提供電子郵件通聯記錄報表，包括郵件進出完整記錄、郵件密錄、郵件備份、郵件搜尋查找功能，郵件統計圖表。

13.WebMail

使用者不需安裝其他軟體就可利用網頁瀏覽器來使用操作郵件系統，Webmail 系統操作簡單方便，就如同使用Outlook一般，不需重新學習！

14.整合協同作業平台

整合協同作業功能，包括郵件、行事曆、全域通訊錄、任務四大功能，提供企業內部溝通共用平台。

15. Outlook整合

提供Full MAPI功能，完整整合Outlook收信軟體，使用者不用重新學習操作。

16.支援手持式裝置同

支援 ActiveSync 協定，不需要另外安裝APP，同步包含郵件、行事曆、通訊錄、工作事項，可大幅提升工作效率。

17.AD帳號整合

提供 AD plugin 功能，可直接在AD管理使用者帳號，也可支援其他系統做單一簽入式認證，當使用者更換新的郵件系統時，不需重新建立所有使用者帳號！

基於開放的 Linux KVM Hypervisor發展與運行。支持Virtual Machine 的生命週期(建立、開機、關機、遷移)管理(VMM,Virtual Machine Management)。以網頁遠距操作，不用進機房就可以維護運算資源。透過 VM Cluster設定與API使用，可讓應用軟體彈性擴充VM資源數量。
