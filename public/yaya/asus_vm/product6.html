  • 專為企業設計的雲端儲存 Gateway 伺服器，獨特的混合儲存雲技術，提供了簡單安全的雲端儲存解決方案，滿足企業檔案同步分享、安全控管、資料保護以及雲端備份的需求。

產品功能與特色

1.各種裝置間的檔案同步

支援 Windows、Mac OS 電腦以及 Android、iOS 行動裝置間的檔案同步，讓企業內部使用者的各個裝置間可以隨時同步重要的檔案。

2.公司內外部的檔案分享

可在公司內部輕鬆分享檔案，解決常常因為檔案太大無法以郵件附件分享的困擾；針對分享檔案修改過的內容也會被即時同步更新到所有使用者。
可使用網站連結的方式分享檔案給外部使用者，還可以設定下載密碼以及有效下載次數、時數，保護重要檔案不外流。

3.企業級的資料安全管控

整合 Windows AD 伺服器，可以輕鬆導入企業現有環境，配合 AD 有效做帳號管控。
可設定使用者分享權限以及空間大小等，亦可察看使用者檔案列表，使用者的檔案存取以及分享紀錄等都也會被詳細記錄。

4.多樣化的檔案存取方式

除了可以使用傳統的網路芳鄰 CIFS 以及 NFS 協定來存取檔案，也可以透過瀏覽器以 HTTP的方式來上傳下載檔案。 另外可以在電腦上安裝 Agent，在智慧型裝置上安裝 App，輕鬆同步存取檔案。

5.完整的資料保護

使用傳統 RAID 1 二顆硬碟保護，確保單顆硬碟損毀系統服務不中斷；所有資料皆會備份到雲端空間，以雲端技術以及專業維護團隊確保資料的安全無虞。支援檔案版本管理以及資料夾快照功能，輕鬆回復因人為或是遭病毒損毀的檔案內容。

6.無限擴充的雲端空間

不同於傳統儲存裝置有空間容量限制，ArkEase 的混合雲架構提供可無限擴充的雲端儲存空間，解決資料量不斷成長的問題。
ArkEase 的本地端儲存所提供之快取功能，更可以讓使用者快速存取所需檔案。

基於開放的 Linux KVM Hypervisor發展與運行。支持Virtual Machine 的生命週期(建立、開機、關機、遷移)管理(VMM,Virtual Machine Management)。以網頁遠距操作，不用進機房就可以維護運算資源。透過 VM Cluster設定與API使用，可讓應用軟體彈性擴充VM資源數量。
