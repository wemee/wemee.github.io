# 願景

* 建立一網頁，可以做數字手寫辨識
* 模型可解釋性：顯示 Top-3 預測機率 + Grad-CAM 熱點圖
* 其他你認為做圖片辨識需要的特點工具

# 技術規格

* 資料集：MNIST（28×28 灰階手寫數字）
* 前端推理框架：TensorFlow.js
* 模型匯出格式：tfjs 格式（從 Keras 轉換）
* 模型權重大小：控制在 500KB 以內
* Canvas 輸入預處理：
  - 筆劃置中與縮放至 28×28
  - 正規化為黑底白字（符合 MNIST 格式）

# task

* 建立一不給 git 追蹤的資料夾，在裡頭先將 model 訓練好
* 使用 python，目標為在此電腦 Macbook M1，10 分鐘內訓練好，95% 準確度以上
* 權重匯出為 TensorFlow.js 格式供前端推理
* 手機版網頁仍然可以做手寫數字推理，所以模型結構需輕量
